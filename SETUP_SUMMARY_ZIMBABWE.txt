================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "No articles found"
SOLUTION:
  1. Check internet connection
  2. Test RSS feeds in browser: https://newsday.co.zw/feed
  3. Check logs: logs/zimbabwe.log
  4. Restart server: python main.py

PROBLEM: "Connection refused" error
SOLUTION:
  1. Ensure server is running: python main.py
  2. Check port 8000 is not in use
  3. Try different port: uvicorn main:app --port 8001

PROBLEM: "All articles filtered out"
SOLUTION:
  1. Reduce filter sensitivity in zimbabwe.yaml:
     min_relevance: 0.3  (was 0.6)
  2. Check if keywords are relevant to your content
  3. Review logs to see which articles are being filtered

PROBLEM: "Social media API not working"
SOLUTION:
  1. Verify token in .env.development hasn't expired
  2. Check API permissions are correct
  3. See SOCIAL_MEDIA_INTEGRATION.md troubleshooting section

PROBLEM: "Database error"
SOLUTION:
  1. Delete news_zimbabwe.db (will recreate on restart)
  2. Check disk space available
  3. Run: python scripts/init_db.py
