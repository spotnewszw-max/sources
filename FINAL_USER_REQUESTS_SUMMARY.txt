üîå API QUICK REFERENCE
================================================================================

BASE URL: http://localhost:8000/api/v1/article-requests

AUTHENTICATION HEADER (Required on all requests):
    X-User-ID: your_user_id

CREATE REQUEST:
    POST /request
    Payload: title, topic, article_type, priority, etc.

ADD THINKING:
    POST /{request_id}/thinking
    Payload: thinking_content, stage, thinking_type, adoption_priority

GENERATE ARTICLE:
    POST /{request_id}/generate
    Returns: article_id, thinking_incorporated_count

GET REQUEST SUMMARY:
    GET /{request_id}/orchestration
    Returns: Full orchestration view with all thinking

1. CREATE REQUEST:
   curl -X POST http://localhost:8000/api/v1/article-requests/request \
     -H "Content-Type: application/json" \

     -H "Content-Type: application/json" \

     -d '{
       "thinking_content":"Include recent RBZ statements",
       "stage":"pre_generation",
       "thinking_type":"suggestion",
       "adoption_priority":8
     }'

3. GENERATE ARTICLE:
   curl -X POST http://localhost:8000/api/v1/article-requests/req_123/generate \
     -H "X-User-ID: analyst_001"

4. GET SUMMARY:
   curl http://localhost:8000/api/v1/article-requests/req_123/orchestration \
     -H "X-User-ID: analyst_001"
